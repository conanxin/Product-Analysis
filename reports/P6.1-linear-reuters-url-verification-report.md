## P6.1 URL 实链验证

### 候选 URL

`https://www.reuters.com/business/atlassian-competitor-linear-raises-funding-125-billion-valuation-2025-06-10/`

### 验证结果

| 测试 | 结果 |
|------|------|
| curl -I (Mozilla UA) | 401 (Datadome) |
| curl -I (Googlebot UA) | 401 (Datadome) |
| web_fetch | timeout / captcha |
| web_search exact title | **未找到** — 通用结果无关 |
| Wayback Machine | reuters.com 通用标题 |
| 标题匹配 | **未确认** — Datadome 阻止标题获取 |

### 评估

- URL 格式与 Reuters article URL pattern 一致（`/business/<slug>-<date>/`）
- HTTP 401 + x-datadome: protected 确认是 Datadome bot 拦截
- 标题未通过任何途径确认
- 故事背景与 TechCrunch 2025 同一天（2025-06-10）一致 — 是同一事件二手转报
- **结论**：URL exists in form, not HTTP-verified, 标 **datadome-protected**
